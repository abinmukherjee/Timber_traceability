⚙️ Admin — Role Management

Grant or revoke Auction House and Factory roles. Only the deployer wallet (DEFAULT_ADMIN_ROLE) can do this.

🔐
How it works: The backend returns the role hash → you sign the grantRole() or revokeRole() transaction in MetaMask → the blockchain emits a RoleGranted/RoleRevoked event → the event listener automatically mirrors the change to the database. No private key is ever stored server-side.
