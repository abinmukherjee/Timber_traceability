# TimberTrace Frontend (React SPA)

Vite + React 18 + React Router + TanStack Query + ethers.js v6 + Tailwind CSS.
Apple-inspired UI. Replaces the old static HTML site (now in `../frontend-legacy/`).

## Why a SPA
- **One persistent wallet connection** across all routes (no more reconnecting on every page).
- **Silent session restore** on load via `eth_accounts` (no MetaMask popup if already approved).
- **Instant navigation** — client-side routing, no full page reloads (fixed the 5–6s delay).
- **Live data** — React Query polls the backend every 5s and refetches immediately after a tx confirms.

## Develop
```bash
cd frontend
npm install
npm run dev        # http://localhost:5500 (or a free port if 5500 is taken)
```
Requires the Spring Boot backend on `:8080`. Override with `VITE_API_BASE` (e.g. in `frontend/.env`).

## Build
```bash
npm run build      # outputs dist/
npm run preview
```

## Contract config
The app fetches `/contract-config.json` (in `public/`) at startup for the address + ABI.
`contracts/scripts/deploy.js` regenerates it on every deploy — swap the file to re-point the
app at a new contract with **no rebuild**.

## Structure
```
src/
  context/WalletContext.jsx   # persistent MetaMask connection
  hooks/                      # useWallet, useQueries (React Query), useScrollReveal
  lib/                        # api.js, contract.js, lastRoute.js
  components/                 # Nav, Layout, DataTable, StatCard, Button, … (Apple-style)
  pages/                      # Dashboard, Auction, Factory, Admin, Verify
public/contract-config.json   # deployed address + ABI (generated by deploy.js)
```

## Docker
`frontend/Dockerfile` builds the app and serves it with nginx (SPA fallback so
`/verify.html?id=` from old QR codes still resolves). `docker-compose.yml` builds this service.
